nhc98 mailing list

List information

This is a list for all users of the nhc98 Haskell compiler, (including implementors, researchers, and ordinary users) to share information, discuss features, report bugs, distribute patches, plan future directions, and so on. We encourage users to submit all bug reports here, rather than directly to the maintainers, to ensure that both the bugs and their fixes are publically accessible via the list archive. You do not need to be a member of the list to search the archive.

Minor releases of the compiler will be announced here. Major releases will also be announced more widely, for instance on the Haskell mailing list.

To subscribe or unsubscribe

To join the list, send a message with the subject subscribe to the address: nhc-users-request@cs.york.ac.uk. To leave the list, send the message unsubscribe to the same address.

To post to the list

Messages intended for the whole list are sent to the address: nhc-users@cs.york.ac.uk

List archive

A web archive of messages from the list is here: http://www.cs.york.ac.uk/fp/nhc98/nhc-users-archive/

The latest updates to these pages are available on the WWW from http://www.cs.york.ac.uk/fp/nhc98/
