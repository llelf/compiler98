nhc98 libraries

Here is the status of nhc98's implementation of the standard libraries defined in the official Haskell '98 Library Report:

library namedescription
Array complete, implementation based on vectors
CPUTime complete
Char complete, except for Unicode support
Complex complete
Directory complete
IO some minor omissions
Ix complete
List complete, with additional fns
Locale complete
Maybe complete
Monad complete, with additional fn `untilM'
Numeric incomplete
Prelude complete
Random complete, reused from Hugs/ghc
Ratio complete
System complete
Time complete except for TimeDiffs

Here is the status of non-standard libraries provided by nhc98:

library namedescription
BinArray imperative binary arrays
Binary binary I/O and binary representations of values
Bit bit-twiddling operations, e.g. and, or, xor
FFI foreign function interface
GreenCard just the standard %dis definitions and constructors you need in order to write your own GreenCard files
Interrupt obsolete
IOExtras like GHC's IOExts
NonStdTrace the common side-effecting trace function
PackedString an efficient representation of strings
Text obsolete

The latest updates to these pages are available on the WWW from http://www.cs.york.ac.uk/fp/nhc98/
