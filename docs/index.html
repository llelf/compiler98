nhc98 from York

nhc98 is a fully-fledged compiler for Haskell 98, the standard lazy functional programming language. It based on Niklas Röjemo's nhc13, a compiler for an earlier version of the language. Written in Haskell, it is small and very portable, and aims to produce small executables that run in small amounts of memory. Is a pattern becoming obvious here? :-)
